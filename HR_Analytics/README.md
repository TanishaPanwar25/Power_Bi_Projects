#  HR Analytics Dashboard - Power BI Project

##  Project Overview
This project focuses on **HR Analytics** using Power BI.  
The objective is to analyze employee data and uncover insights related to workforce trends, employee performance, and organizational growth.

---

##  Folder Structure
HR_Analytics/

├── 0.jpg - 2.jpg                     # Dashboard screenshots  
├── Dates.xlsx                       # Date dimension data  
├── hR_Analytics.pbix                # Power BI dashboard file  
├── people_data.csv                  # Employee basic details  
├── people_employment_history.csv    # Employment history data  

---

##  Tools & Technologies Used
- Power BI  
- Microsoft Excel  
- CSV Data Sources  
- Data Visualization Techniques  

---

##  Key Features of Dashboard
- Employee distribution analysis  
- Attrition & retention insights  
- Department-wise performance  
- Hiring and employment trends  
- Interactive filters and slicers  

---

##  Insights Generated
- Identified employee attrition patterns  
- Analyzed hiring trends over time  
- Highlighted department-wise workforce distribution  
- Improved understanding of employee lifecycle  

---

##  How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Load data if required  
4. Explore interactive dashboards
